# 题目描述
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0，第1项是1）。
n<=39
# 解析
```c#
n=0时，返回0
n=1时，返回1
否则，使用size为3的数组，前面2个相加等于第三个，循环加
```
# 代码
```c#
class Solution
{
    public int Fibonacci(int n)
    {
        // write code here
        int[] re = new int[3];
        if(n==0) 
            return 0;
        else if(n==1)
            return 1;
        else{
            re[0]=0;
            re[1]=1;
            for(int i=0;i<n;i++){
                re[2] = re[0] + re[1];
                re[1] = re[0];
                re[0] = re[2];
            }
            return re[2];
        }                        
    }
}
```
